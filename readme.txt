Данный блог был написан с целью создания общего информационанного пространства о животных, в которых
пользователи могут обмениться всей необходимой информацией. В дальнейшем блог будет разделён на секции: здоровье,
дрессировка, питание и т.д.

Для запуска проекта необходимы pip и Django 4.0.
При использовании python 3.4 и выше pip устанавливается вместе с python автоматически.

Для установки Django в терминал необходимо ввести следующую команду:
  $ pip install Django
